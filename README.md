## Overview 🌠
Este é um projeto python que visa automatizar o processo de classificação de clientes com base no seu score afim de gerar um relatório para a equipe de análise de crédito.


## Scripts Importantes 📃
- Arquivo principal: [main.py](main.py)
- Geração de dados [gerar_dados.py](./scripts/gerar_dados.py)

## Como executar ✅

1. Clone o repositório do git
2. Entre nele e crie um venv
3. Instale todas as bibliotecas do requirements.txt
4. Rode o script `main.py`
